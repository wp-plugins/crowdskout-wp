Easily adds the Crowdskout analytics suite to all client websites.

== Description ==

With a simple installation, this plugin adds the [Crowdskout](http://crowdskout.com) Analytics JavaScript to the footer of any WordPress site. After entering the source ID that can be found in the Crowdskout admin area, all page views will be ingested in your site for analysis with the Crowdskout platform.

== Installation ==

* Login to the Crowdskout platform, and navigate to Settings > Web Site and note the Source ID.
* Now login to the WordPress backend, and navigate to Settings > Crowdskout. Once you enter your Source ID here, your page views will automatically start tracking.
